# Connections Guide

Complete guide to networking and connections on Spotter.

## Overview

Connections are the foundation of Spotter networking. Build your golf network by connecting with other members you know or want to play with.

## Connection Basics

### What Are Connections?

- **Mutual agreement** between two members to connect
- Visible in each other's network
- Can view full profiles (within tier visibility)
- Can message each other (in-app)

### Connection Limits by Tier

| Tier | Max Connections |
|------|-----------------|
| FREE | 50 |
| SELECT | 500 |
| SUMMIT | Unlimited |

### Same-Tier Visibility

Spotter enforces **same-tier visibility**:
- You can only send connection requests to members in your tier or lower
- Members in higher tiers can see you (but you can't see them unless you upgrade)
- This ensures quality matching within similar investment levels

Example:
- You're FREE → Can connect with FREE members
- You're SELECT → Can connect with FREE and SELECT
- You're SUMMIT → Can connect with any tier

## Sending Connection Requests

### From Member Profile

1. Find member you want to connect with
2. Tap "Connect" button
3. Select **connection type**:
   - **Played Together** - You've golfed together
   - **Introduced** - Someone introduced you
   - **Met Offline** - Met in person (not on course)
   - **Online Only** - Only interacted online
4. Add optional personal message
5. Tap "Send Request"

### From Round Participants

After playing a round together:
1. Go to completed round
2. View participant list
3. Tap "Connect" next to players you met
4. Connection type auto-set to "Played Together"

### From Event Attendees

1. Go to completed event
2. View attendee list
3. Tap "Connect" on profiles
4. Select appropriate connection type

## Managing Incoming Requests

### Viewing Requests

1. Go to **Network** tab
2. Tap **Requests** section
3. See Incoming and Outgoing tabs

### Responding to Requests

**To Accept:**
1. View request details
2. Review their profile
3. Tap "Accept Connection"
4. Now connected - appear in each other's networks

**To Decline:**
1. View request details
2. Tap "Decline"
3. Optional: Add note why
4. Requester not notified (just sees "declined" status)

**To Block:**
1. View request
2. Tap "Block"
3. Prevents future requests from this person

## Connection Types Explained

### Played Together

- **Meaning:** Actually golfed together on a course
- **When to use:** After a round, scramble, or tournament
- **Benefit:** Highest credibility, best for reputation

### Introduced

- **Meaning:** Someone you both know made the introduction
- **When to use:** Friend/connection vouched for someone
- **Benefit:** Warm connection with mutual reference

### Met Offline

- **Meaning:** Met in person but not on golf course
- **When to use:** Business meeting, social event, etc.
- **Benefit:** Personal connection established

### Online Only

- **Meaning:** Only interacted via Spotter app/messages
- **When to use:** Haven't met in person yet
- **Benefit:** Start building relationship

**Tip:** Choose the most accurate type. "Played Together" carries most weight for reputation.

## Building Your Network

### Quality vs. Quantity

**Focus on quality connections:**
- People you've actually played with
- Members you want to play with again
- Golfers with similar skill level
- Potential business connections

**Avoid:**
- Random connection spam
- Connecting with everyone you see
- Fake connection types

### Network Building Strategies

**For FREE Members (50 connections):**
1. Be selective - these are your core network
2. Connect after every round you play
3. Prioritize people you want to play with again
4. Remove inactive connections to make room

**For SELECT Members (500 connections):**
1. Build regional network
2. Connect with members at your home course
3. Network at every event you attend
4. Use introductions to expand reach

**For SUMMIT Members (Unlimited):**
1. Build extensive network
2. Connect with other SUMMIT members
3. Focus on quality business connections
4. Leverage Priority Boosts for visibility

### Finding Members to Connect With

**Golf Tab:**
- Join rounds → Meet new people → Connect after

**Network Tab:**
- Discover section → Browse members → Filter by location, handicap, etc.

**Events:**
- Attend tournaments → Meet attendees → Connect after

**Introductions (SELECT+):**
- Ask mutual connections for introductions
- Introduction requests boost reputation

## Introductions (SELECT+)

### What Are Introductions?

When two of your connections aren't connected, you can introduce them:
- Facilitates warm connections
- Builds your reputation
- Helps your network grow

### Sending Introductions

1. Go to Connection A's profile
2. Tap "Introduce to..."
3. Select Connection B
4. Add message explaining why they should connect
5. Send introduction request

Both parties see:
- Your message
- Each other's profiles
- Option to accept/decline the connection

### Receiving Introductions

1. Get notification: "John wants to introduce you to Jane"
2. View Jane's profile
3. Read John's introduction message
4. Accept or decline

### Introduction Credits

- SELECT: 2 introduction credits per month
- SUMMIT: Unlimited introductions

Credits reset monthly for SELECT tier.

## Managing Your Connections

### Connection List

**Network tab shows:**
- All connections (alphabetical)
- Recent activity
- Connection strength

### Filtering & Sorting

Filter by:
- Location (city, state)
- Handicap range
- Industry (professional identity)
- Connection type
- Date connected

Sort by:
- Recently connected
- Alphabetical
- Handicap (low to high)
- Reputation score

### Connection Details

Tap any connection to see:
- Full profile (tier visibility permitting)
- Common connections
- Rounds played together
- Message history

### Removing Connections

1. Go to connection's profile
2. Tap "..." menu
3. Select "Remove Connection"
4. Confirm

**Effects:**
- No longer appear in each other's networks
- Can send new request in future
- Past round history remains

## Reputation Impact

Connections affect your reputation score:

**Positive:**
- Accepting connections (+2 points)
- Making introductions (+15 points)
- Large quality network (+20 points at 100+ connections)

**Negative:**
- Frequently declining connections (-5 points)
- Removing connections (-3 points)
- Fake connection types reported (-20 points)

## Troubleshooting

### Can't send connection request

**Possible reasons:**
- At connection limit (upgrade tier)
- User in higher tier (can't see them)
- Already connected
- Previous request pending
- User blocked you

### Request not accepted

- Give them time (busy golfers)
- Follow up with message (if connected another way)
- Don't spam multiple requests
- Focus on people you've actually met

### Connection disappeared

- Other user removed connection
- Account deactivated
- Same-tier visibility changed (you or they upgraded)

### Can't see certain members

- They're in higher tier (upgrade to see)
- Profile set to private
- Geographic filters active
- Same-tier visibility restrictions

## FAQ

**Q: Can I message someone without connecting?**
A: SUMMIT members can message same-tier members directly. Others need to connect first.

**Q: What if someone sends inappropriate request?**
A: Block them. Report if serious. Go to request → "..." → Block.

**Q: Do declined requests count against limits?**
A: No, declined requests don't count toward your connection limit.

**Q: Can I change connection type after connecting?**
A: Yes, edit connection details in their profile.

**Q: Why can't I see some of my connections' profiles?**
A: They may have upgraded to higher tier. Upgrade to see them again.

**Q: How do I know who to connect with?**
A: People you've played with, met at events, or have mutual connections with.
