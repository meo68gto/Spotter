# Tier Upgrade Guide

Guide to upgrading your Spotter membership tier (FREE → SELECT → SUMMIT).

## Overview

Spotter offers three membership tiers for golfers:

| Feature | FREE | SELECT | SUMMIT |
|---------|------|--------|--------|
| Cost | Free | $1,000/year | $10,000 lifetime |
| Member Discovery | 20 results | Unlimited | Unlimited |
| Connections | 50 | 500 | Unlimited |
| Golf Rounds | Join only | Create 4/month | Unlimited |
| Coaching | Limited | Full access | Full access |
| Priority Boosts | ❌ | ❌ | ✅ |

## FREE Tier

**Cost:** $0

**Included:**
- Basic profile (name, photo, bio, location)
- View up to 20 member search results
- Up to 50 connections
- Join public golf rounds
- View public events
- Basic reputation tracking

**Best for:** Casual golfers testing the platform

## SELECT Tier

**Cost:** $1,000/year

**Includes everything in FREE, plus:**
- Extended profile (professional + golf identity)
- Unlimited member discovery
- Up to 500 connections
- Create up to 4 rounds per month
- Send introduction requests
- Book coaching sessions
- Register for all events
- Email support

**Best for:** Active golfers building their network

## SUMMIT Tier

**Cost:** $10,000 one-time (lifetime membership)

**Includes everything in SELECT, plus:**
- Unlimited connections
- Unlimited round creation
- Priority Boosts (appear first in searches)
- Early event registration
- Priority support
- Exclusive SUMMIT-only events
- Lifetime membership (never expires)

**Best for:** Serious golfers, business networking

## How to Upgrade

### In-App Upgrade

1. Go to **Profile** tab
2. Tap **Current Tier** (e.g., "FREE")
3. Select **SELECT** or **SUMMIT**
4. Review upgrade details
5. Tap **Upgrade Now**
6. Complete payment via Stripe Checkout
7. Return to app - your tier updates automatically

### Web Upgrade

1. Visit https://spotter.golf/upgrade
2. Sign in to your account
3. Choose tier and complete payment
4. Changes apply to mobile app immediately

## Payment Methods

Accepted payment methods:
- Credit/debit cards (Visa, Mastercard, Amex, Discover)
- Apple Pay (in-app)
- Google Pay (in-app)

All payments processed securely via Stripe.

## Upgrade Paths

### FREE → SELECT

**Prerequisites:**
- Active Spotter account
- Valid email address

**Process:**
1. Initiate upgrade
2. Enter payment details
3. Annual billing starts immediately
4. New features available immediately

### FREE → SUMMIT

**Prerequisites:**
- Active Spotter account
- Valid email address

**Process:**
1. Initiate upgrade
2. Enter payment details
3. One-time payment
4. Lifetime membership activated
5. All features available immediately

### SELECT → SUMMIT

**Prerequisites:**
- Active SELECT membership

**Process:**
1. Initiate upgrade
2. Pay prorated difference (if upgrading mid-year)
3. SELECT billing stops
4. SUMMIT lifetime membership activated

## Billing Details

### SELECT Tier Billing

- **Billing cycle:** Annual
- **Charge date:** Day you upgraded
- **Renewal:** Automatic annual renewal
- **Receipt:** Email receipt sent after each charge

### Managing Billing

1. Go to Profile → Settings → Subscription
2. View:
   - Current tier
   - Next billing date
   - Payment method
   - Billing history

### Canceling SELECT

1. Go to Profile → Settings → Subscription
2. Tap **Cancel Subscription**
3. Confirm cancellation
4. Current period continues until end date
5. Reverts to FREE tier after expiration

**Note:** SUMMIT tier is lifetime - no cancellation needed.

## What Happens After Upgrade

### Immediate Changes

- Tier badge updates in app
- New features become available
- Profile sections unlock
- Connection limits increase
- Round creation enabled (SELECT+)

### Data Migration

Your data stays intact:
- Existing connections remain
- Profile information preserved
- Reputation score continues
- Event registrations stay valid

### Downgrade Considerations

If you downgrade from SELECT to FREE:
- Keep existing connections (even if over 50 limit)
- Cannot add new connections if over limit
- Round creation disabled
- Extended profile hidden from others
- Coaching bookings remain active

## Frequently Asked Questions

### Payment & Billing

**Q: Is my payment secure?**
A: Yes. All payments processed via Stripe, PCI compliant. Spotter never stores your card details.

**Q: Can I get a refund?**
A: SELECT: Prorated refund within 14 days. SUMMIT: No refunds (clearly stated at purchase).

**Q: What happens if my card expires?**
A: You'll receive email reminders. If payment fails, tier status changes to "past_due" for 7 days before reverting to FREE.

**Q: Can I switch from annual to monthly?**
A: SELECT is annual only. SUMMIT is one-time lifetime.

### Features

**Q: Do I need SELECT to book coaching?**
A: Yes, coaching requires SELECT or SUMMIT tier.

**Q: Can I create rounds as a FREE member?**
A: No, round creation requires SELECT+ tier. You can join rounds created by others.

**Q: What's a Priority Boost?**
A: SUMMIT members appear higher in member search results, increasing visibility.

**Q: Can I gift a membership?**
A: Coming soon. Currently each user must upgrade their own account.

### Technical

**Q: How long does upgrade take?**
A: Immediate after payment completes. Restart app if not reflected within 1 minute.

**Q: What if upgrade fails?**
A: Check payment method. Contact support@spotter.golf with error message.

**Q: Can I use upgrade on multiple devices?**
A: Yes, your tier is tied to your account, not device.

## Troubleshooting

### Payment declined

1. Check card details
2. Ensure sufficient funds
3. Contact bank if persistent
4. Try different payment method

### Upgrade not reflecting

1. Force close and reopen app
2. Check internet connection
3. Log out and back in
4. Contact support if persists

### Double charged

1. Check email for receipts
2. Contact support@spotter.golf
3. Include transaction IDs
