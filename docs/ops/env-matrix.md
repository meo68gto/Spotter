## Ownership

- Platform owner: backend lead (`Supabase`, `Stripe`, `Sentry`, `PostHog` secrets)
- Mobile owner: mobile lead (`EXPO_PUBLIC_*` runtime config)
- Compliance owner: legal/ops (`LEGAL_*` versions and URLs)
