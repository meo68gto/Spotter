# Mobile Store Assets Spec

## Ownership
- Product copy owner:
- Design owner:
- Capture owner:
- Upload owner:

## iOS App Store Connect assets
1. App icon: 1024x1024 PNG (no alpha).
2. iPhone screenshots:
- 6.7" display set
- 6.5" display set (if required)
3. iPad screenshots (if iPad supported for listing).
4. Optional app preview videos.

## Google Play assets
1. High-res icon: 512x512 PNG.
2. Feature graphic: 1024x500 PNG.
3. Phone screenshots (min 2, max 8).
4. 7-inch and 10-inch tablet screenshots (if tablet listing used).

## Copy checklist
1. App subtitle / short description.
2. Full description.
3. Keywords / tags.
4. Privacy policy URL and legal links.
5. Permissions justification notes.

## Naming convention
- `<platform>_<locale>_<surface>_<index>.png`
- Example: `ios_en-US_home_01.png`

## Required evidence before submission
- `apps/mobile/assets/store/manifest.json` completed.
- Capture SOP executed and archived.
- Final asset package reviewed by product + design.
